# План управления проектом — FinQuest

**Проект:** FinQuest — геймифицированный AI-ассистент для личных финансов  
**Исполнитель:** Осипов Матвей Васильевич, Виноградов Алексей Олегович  
**Университет:** РЭУ им. Г.В. Плеханова  

---

## Цели проекта

1. Разработать веб-приложение для учёта личных финансов с элементами геймификации
2. Интегрировать AI-советника на основе LLM (Claude Haiku / Gemini Flash)
3. Реализовать полный стек: Go-бэкенд, React-фронтенд, PostgreSQL, Docker
4. Подготовить документацию и отчёт о тестировании

---

## Хронология разработки

Фактическое время разработки: **6 дней** (от согласования требований до финальной реализации). Каждый этап соответствует одному рабочему дню. Вся команда работала параллельно: backend (Осипов) + аналитика/frontend/документация (Виноградов).
