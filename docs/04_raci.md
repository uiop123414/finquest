# Матрица ответственности RACI — FinQuest

## Условные обозначения

| Буква | Роль | Описание |
|-------|------|----------|
| **R** | Responsible | Исполнитель — выполняет задачу |
| **A** | Accountable | Ответственный — отвечает за результат |
| **C** | Consulted | Консультируемый — даёт экспертизу |
| **I** | Informed | Информируемый — получает уведомление |
